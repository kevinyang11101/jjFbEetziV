# 宠物领养管理系统 java1120

## 项目概述

宠物领养管理系统是一款基于前后端分离架构的B/S系统，主要用于管理宠物领养相关的各项事务，包括流浪动物的救助、领养申请处理、宠物信息管理以及社区互动等功能。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows/Mac
- 开发环境：
  - IDEA
  - JDK1.8
  - Maven
  - Mysql
- 技术实现：
  - 后端：Java、SpringBoot、Mybatis
  - 前端：Vue
  - 数据库：Mysql

## 功能模块

### 系统角色

#### 管理员

- 前台操作：涵盖用户所有操作
- 后台管理：
  - 用户管理
  - 角色管理
  - 菜单管理
  - 文件管理
  - 流浪动物管理
  - 待绝育动物管理
  - 申请领养管理
  - 评论管理
  - 流浪动物救助
  - 喂养点管理
  - 动物走失管理
  - 救助站管理
  - 帖子管理
  - 捐赠管理
  - 公告管理
  - 科普文章管理
  - 活动管理

#### 用户

- 首页浏览
- 宠物领养
- 流浪宠物救助
- 宠物喂养点查询
- 丢失宠物查看
- 流浪宠物救助站信息查询
- 宠物论坛互动
- 宠物捐赠
- 科普文章阅读
- 热门活动参与
